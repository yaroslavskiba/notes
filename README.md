Приложение для заметок.
  Реализованный функционал: 
   - добавление заметки;
   - удаление заметки;
   - редактирование заметки;
   - фильтрация по тегу;
   - удаление тега с определенной заметки;

  Технологии: 
   - React;
   - Typescript;
   - Redux Toolkit;

   Установка:
   
    git clone git@github.com:yaroslavskiba/notes.git
    cd notes
    npm i
